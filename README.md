# **Получение списка игр**

Этот проект сделал для быстрого получения списка игр

## **Как установить**

Для начало получите ключ - RAWG токен, для работы с программой. Как и где это сделать вы найдёте **[тут](https://rawg.io/apidocs)** 

Далее в папке с проектом создайте файл `.env` и вставьте туда
`TOKEN="Ваш Holiday API токен"`

Python3 должен быть уже установлен. Затем используйте `pip` (или `pip3`, есть конфликт с Python2) для установки зависимостей:

```pip install -r requirements.txt```

Рекомендуется использовать [virtualenv/venv](https://docs.python.org/3/library/venv.html) для изоляции проекта

## **Как запустить программу**

1. Запускаем командную строку (`Win+R >>> CMD`)
2. Перемещаемся в директорию с программой (`cd <путь к файлу>`)
3. Запускаем файл программы. (`python main.py`)

Если вы хотите получить список игр по своим параметрам, то в файле `main.py` на строке `10` вы можете заменить параметры в переменной `payload` на свои из **[документации](https://api.rawg.io/docs/)** 

### **Возможные ошибки**

Программа не будет работать с неверным API токеном и с неверными параметрами `payload`
